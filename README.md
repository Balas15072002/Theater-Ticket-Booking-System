# Theater-Ticket-Booking-System
Project: Theater Ticket Booking System
Title: ShowTime
Target Audience: Fans, families, friends, cinemaholics, Theater Owners, Theater Lovers, Theatre Seat Managers, Movie Enthusiasts 
Type: B2C
Timeline: 3 weeks(approximately)
Budget: 5 lakhs
Total Members: 5 Members

Feature:

	---Theatre Seat Manager---
	
	-Register & Login Pages
	-Secure login for Theatre Seat Manager with role-based access control
	-Book tickets for users (with UPI integration: GPay, PhonePe, etc.)
	-Cancel tickets
	-Receive notifications when users book tickets
	-View and manage user details
	-Modify ticket prices dynamically (increase/decrease) 
	-Add discount pricing
	-Close bookings manually for sold-out or blocked shows
	-View seat availability in real-time
	-Search & filter movies by title, genre, or location 
	-View detailed movie info (cast, singer, release date, etc.)
	
	---User---

	-Separate Register & Login Pages for users
	-Book tickets using secure UPI payment (GPay, PhonePe, etc.)
	-Cancel tickets
	-View seat details (available/unavailable)
	-Search & filter movies by location or title
	-View movie details including cast, singers, genre, etc
